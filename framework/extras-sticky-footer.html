##Sticky Footer Remember the natural stacking order? HTML elements stack on top of each other in the order that they appear in the HTML. Each element is as tall as their content (unless otherwise specified). On the home page, the footer appears right below the headings. On the about page, it appears on the bottom because the content is long enough to scroll. To ensure that your footer is always at the bottom of the page, no matter how long the page content is or how big the browser window is, use a "sticky footer".
##Create a Sticky Footer We'll be following these instructions: http://css-tricks.com/snippets/css/sticky-footer/ First, we need to create a page wrapper around the page content only, **not** the footer content. The home page HTML will look like similar to this: ``` <div class="page-wrap"> <header> <h1>Jane Smith</h1> <h2>Web Developer + City Girl</h2> <nav> ... </nav> </header> </div><!-- close .page-wrap --> <footer> ... </footer> ``` The about and contact pages will look like this: ```xml <div class="page-wrap"> <header> <h1>Jane Smith</h1> <h2>Web Developer + City Girl</h2> <nav> ... </nav> </header> <section> ... </section> </div><!-- close .page-wrap --> <footer> ... </footer> ```
##Sticky Footer CSS ``` /* sticky footer */ .page-wrap { min-height: 100%; margin-bottom: -35px; /* must be the same height as footer */ } .page-wrap:after { content: ""; display: block; } footer, .page-wrap:after { height: 35px; /* wowever tall you want the footer to be */ } footer { background: #222; color: #ccc; } ``` **NOTE**: In the tutorial this lesson is based on, you will see the following CSS included but we've already added it to the CSS when creating the home page background image. ``` html, body { height: 100%; } ```
##One more update Let's revise the CSS for the paragraph in the footer. ``` footer { /* add this to the existing declaration block */ clear: both; padding: 5px; /* adds space inside the footer */ } footer p { margin: 0; /* removes default margin from paragraph */ } ``` Compare your page with this [example](../project/examples/sticky-footer-home.html).
